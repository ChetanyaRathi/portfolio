# Personal Portfolio Website - Chetanya Rathi


This is the source code for my personal portfolio website, built from the ground up to showcase my skills, professional experience, and projects. The site is a fully responsive single-page application built with React, featuring a modern design, interactive animations, and a rule-based AI assistant.

**🔗 Live Demo:** [**https://chetanyarathi-portfolio.netlify.app/**](https://chetanyarathi-portfolio.netlify.app/)
---

## ✨ Features

-   **Modern & Responsive Design:** The UI is fully responsive and looks great on all devices, from mobile phones to desktops.
-   **Interactive UI:** Features smooth animations on scroll, hover effects, a dynamic text rotator, and revolving icons to create a professional and engaging user experience.
-   **Rule-Based AI Assistant:** A custom-built chatbot that can answer questions about my skills, experience, and projects based on a predefined knowledge base.
-   **Component-Based Architecture:** Built with React, the code is organized into reusable and maintainable components for each section of the portfolio.

---

## 🛠️ Tech Stack

-   **Frontend:** React, Vite, HTML5, CSS3
-   **Styling:** Pure CSS with CSS Variables for easy theming and animations.
-   **Deployment:** Git, GitHub, and Netlify for a seamless CI/CD (Continuous Integration/Continuous Deployment) pipeline.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm installed on your machine.
- You can download them from [nodejs.org](https://nodejs.org/).

### Installation & Setup

1.  **Clone the repository:**
    ```sh
    git clone [https://github.com/ChetanyaRathi/portfolio-react.git](https://github.com/ChetanyaRathi/portfolio-react.git)
    ```
    2.  **Navigate into the project directory:**
    ```sh
    cd portfolio-react
    ```

3.  **Install NPM packages:**
    ```sh
    npm install
    ```

4.  **Run the development server:**
    ```sh
    npm run dev
    ```
    The app will be running at `http://localhost:5173/` (or the next available port).
